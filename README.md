# Food Delivery System Analysis (MySQL)

## Project Overview

This project analyzes a food delivery system database using SQL.  
The objective is to understand customer behavior, order trends, revenue generation, and delivery performance.

## Database Tables

• Customers  
• Restaurants  
• Menu  
• Orders  
• Order Details  
• Delivery Partners  
• Payments  
• Feedback  

## Key Analysis

• Customer-wise Order Analysis  
• Restaurant-wise Order Count  
• Total Revenue from Completed Orders  
• Highest Spending Customer  
• Most Ordered Items  
• Average Order Value  
• City-wise Customer Distribution  
• Delivery Partner Performance  
• Failed Payment Analysis  
• Customer Feedback & Ratings  

## Tools Used

• MySQL  

## Key Insights

• Pune city has a high number of customers  
• Completed orders generate major revenue  
• Some orders are cancelled due to various reasons  
• Certain items are ordered more frequently  
• Customer feedback helps evaluate service quality
